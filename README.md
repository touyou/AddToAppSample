# Add-to-app Sample

Add-to-app を活用した実験的サンプルアプリです。
ナビゲーション部分を SwiftUI で実装し、コンテンツ部分を Flutter で実装しています。

## 構成

- `App`: iOS アプリ本体
- `flutter_module`: Flutter モジュール
- `docs`: ドキュメント
